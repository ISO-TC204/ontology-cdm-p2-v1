# Contact Pattern

This ontology specifies the city-level concepts for the contact pattern of the city data model.

This pattern imports the following patterns:

- [City Pattern](CityPattern.md)

This pattern consists of the following classes:

- [Address](Address.md)
- [Address Type](AddressType.md)
- [Contact Thing](ContactThing.md)
- [Country](Country.md)
- [Phone Number](PhoneNumber.md)
- [Phone Type](PhoneType.md)
- [State](State.md)
- [Street Direction](StreetDirection.md)
- [Street Type](StreetType.md)

The formal definition of this pattern is available in [OWL Syntax](ContactPattern.owl).
