# Bylaw Pattern

This ontology specifies the city-level concepts for the bylaw pattern of the city data model.

This pattern imports the following patterns:

- Generic Properties Pattern
- [Organization Pattern](OrganizationPattern.md)
- [Person Pattern](PersonPattern.md)
- v1

This pattern consists of the following classes:

- [Amending Bylaw](AmendingBylaw.md)
- [Bylaw](Bylaw.md)
- [Bylaw Thing](BylawThing.md)
- [Clause](Clause.md)
- [Definition](Definition.md)
- [Jurisdictional Area (BylawPattern)](JurisdictionalArea.md)
- [Law](Law.md)
- [Main Bylaw](MainBylaw.md)
- [Revision Bylaw](RevisionBylaw.md)
- [Schedule](Schedule.md)

The formal definition of this pattern is available in [OWL Syntax](BylawPattern.owl).
