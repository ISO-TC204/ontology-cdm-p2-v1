# Information technology - City data model - Part 2: City level concepts - Bylaw Pattern

This ontology specifies the city-level concepts for the bylaw pattern of the city data model.

This pattern imports the following files:

- [https://w3id.org/citydata/21972/v1/](https://w3id.org/citydata/21972/v1/)
- [https://w3id.org/citydata/part1/v1/GenericPropertiesPattern](https://w3id.org/citydata/part1/v1/GenericPropertiesPattern)
- [https://w3id.org/citydata/part2/v1/OrganizationPattern](https://w3id.org/citydata/part2/v1/OrganizationPattern)
- [https://w3id.org/citydata/part2/v1/PersonPattern](https://w3id.org/citydata/part2/v1/PersonPattern)

This pattern consists of the following classes:

- [Amending Bylaw](AmendingBylaw.md)
- [Bylaw](Bylaw.md)
- [Bylaw Thing](BylawThing.md)
- [Bylaw Type Code](BylawTypeCode.md)
- [Clause](Clause.md)
- [Definition](Definition.md)
- [Jurisdictional Area](JurisdictionalArea.md)
- [Law](Law.md)
- [Legislation Legal Force Code](LegislationLegalForceCode.md)
- [Main Bylaw](MainBylaw.md)
- [Revision Bylaw](RevisionBylaw.md)
- [Schedule](Schedule.md)
