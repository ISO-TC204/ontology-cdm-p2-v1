# Information technology - City data model - Part 2: City level concepts - Person pattern

This ontology specifies the city-level concepts for the person pattern of the city data model.

This pattern imports the following files:

- [https://w3id.org/citydata/part2/v1/ContactPattern](https://w3id.org/citydata/part2/v1/ContactPattern)

This pattern consists of the following classes:

- [Citizenship](Citizenship.md)
- [Education](Education.md)
- [Gender](Gender.md)
- [Id Type](IDType.md)
- [Name](Name.md)
- [Person](Person.md)
- [Person Id](PersonId.md)
- [Person Name](PersonName.md)
- [Person Thing](PersonThing.md)
- [Sex](Sex.md)
- [Skill](Skill.md)
