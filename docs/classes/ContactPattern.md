# Information technology - City data model - Part 2: City level concepts - Contact pattern

This ontology specifies the city-level concepts for the contact pattern of the city data model.

This pattern imports the following files:

- [https://w3id.org/citydata/part2/v1/CityPattern](https://w3id.org/citydata/part2/v1/CityPattern)

This pattern consists of the following classes:

- [Address](Address.md)
- [Address Type](AddressType.md)
- [Contact Thing](ContactThing.md)
- [Country](Country.md)
- [Phone Number](PhoneNumber.md)
- [Phone Type](PhoneType.md)
- [State](State.md)
- [Street Direction](StreetDirection.md)
- [Street Type](StreetType.md)

- [AddressDataProperty](../properties/AddressDataProperty.md)

- [hasAreaCode](../properties/hasAreaCode.md)
