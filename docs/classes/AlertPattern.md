# Information technology - City data model - Part 2: City-level concepts - Alert pattern

This ontology specifies the city-level concepts for the alert pattern of the city data model.

This pattern imports the following files:

- [https://w3id.org/citydata/part2/v1/CodePattern](https://w3id.org/citydata/part2/v1/CodePattern)

This pattern consists of the following classes:

- [Alert](Alert.md)
- [Alert Thing](AlertThing.md)
This module defines the following properties:

- [activities](../properties/activities.md)
- [AlertDataProperty](../properties/AlertDataProperty.md)
- [AlertObjectProperty](../properties/AlertObjectProperty.md)
- [beginInstant](../properties/beginInstant.md)
- [category](../properties/category.md)
- [cause](../properties/cause.md)
- [certainty](../properties/certainty.md)
- [creationInstant](../properties/creationInstant.md)
- [distribution](../properties/distribution.md)
- [effect](../properties/effect.md)
- [endInstant](../properties/endInstant.md)
- [expiryInstant](../properties/expiryInstant.md)
- [headline](../properties/headline.md)
- [issuer](../properties/issuer.md)
- [lastUpdateInstant](../properties/lastUpdateInstant.md)
- [location](../properties/location.md)
- [recommendation](../properties/recommendation.md)
- [referencedAlert](../properties/referencedAlert.md)
- [resources](../properties/resources.md)
- [severity](../properties/severity.md)
- [status](../properties/status.md)
- [urgency](../properties/urgency.md)
- [website](../properties/website.md)


The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../AlertPattern.ttl) and the SHACL [restrictions](../AlertSHACL.ttl).
