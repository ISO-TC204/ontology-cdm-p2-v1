# Alert Pattern

This ontology specifies the city-level concepts for the alert pattern of the city data model.

This pattern imports the following patterns:

- [Code Pattern](CodePattern.md)

This pattern consists of the following classes:

- [Alert](Alert.md)
- [Alert Thing](AlertThing.md)

The formal definition of this pattern is available in [OWL Syntax](AlertPattern.owl).
